# Our Services

Welcome to **Tech Station Tools & API Services**.  
Here you will find all the powerful tools we provide for developers, students, and content creators.

---

## 1. Blur Page Tool

Instantly blur sensitive text, paragraphs, or documents.

### **Features**

- Blur specific words or entire text
- 3-level blur (low, medium, high)
- Secure and private
- API + UI both supported

### **Use Cases**

- Hide email, phone, or ID
- Create blur effects for tutorials
- Protect sensitive client data

---

## 2. JSON Search Generator

Generate structured and searchable JSON data from any topic.

### **Features**

- AI-powered JSON generation
- Creates optimized title, description, keywords
- Great for search systems and documentation
- Highly customizable

### **Use Cases**

- Search engine data
- Knowledge base
- Filterable content

## 3. Code to Image Generator

Turn any code block into a beautiful image.

### **Features**

- Multiple themes (dark, light, midnight)
- Supports 20+ languages
- Export as PNG/JPG
- Perfect for tutorials, blogs, and social media

### **Use Cases**

- YouTube tutorials
- Programming blogs
- Portfolio content
